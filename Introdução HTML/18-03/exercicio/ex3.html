Perguntas e respostas

Perguntas:

1: Qual é o significado das entidades &lt;, &gt; e &amp;?

2: Para que serve o atributo href da tag <a>?

3: Qual é a diferença entre endereço relativo e endereço absoluto?

4: Para que serve o atributo charset da tag <meta>?















































Respostas:

R1: &lt; representa o caracter <, &gt; representa o caracter > e &amp; represena &.

Topo













































R2: HREF serve para especificar o destino do link.

Topo













































R3: O endereço absoluto é a informação para a localização do recurso de forma independente da página atual (protocolo + domínio + caminho). O endereço relativo é a informação para a localização do recurso a partir da página atual.

Topo













































R4: O atributo charset=”UTF-8″ é um atributo utilizado para comunicar aos navegadores qual é o formato de codificação de caracteres utilizado naquele documento.

Topo
