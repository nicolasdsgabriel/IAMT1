Alguns elementos HTML e seus significados

Elemento 1

<head>: A tag head representa o cabeçalho do documento, onde se é definifa informações como o titulo da página no navegador.

Elemento 2

<body>: A tag body separa a informações do corpo do documento.

Elemento 3

<P>: A tag P (parágrafo) serve para definir os parágrafos do documento.

Elemento 4

<section>: Serve para agrupar conteúdos que sejam semanticamente relacionados.
