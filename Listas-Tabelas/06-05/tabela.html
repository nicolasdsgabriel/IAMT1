Conceitos em HTML Significado
Elementos Elemento img Este elemento permite inserir imagem em uma página HTML. Utiliza-se a tag <img/> (tag vazia). Para toda imagem, deve-se definir suas dimensões (altura e largura) e uma descrição (atributo alt), que será exibida, caso a imagem não possa ser vizualizada.
Elemento table tag table A tag table permirte iniciar a definição de uma tabela
tag tr a tag tr permite difinir cada linha de uma tabela
tag td A tag td permite definir cada célula de conteúdo em uma tabela. O elemento td somente pode ser específico no contexto do elemento.
